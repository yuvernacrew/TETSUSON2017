隣人愛　Secure Base

うさぎを模した心の隣人“neighbor”。人間でも何ものでもないナニカ。
ただ寄り添ってくれる存在。誰もが一人ぼっちで泣かないように、neighborはいる。
この優しい存在に気づいたなら、心に住まわせたなら、誰もが心に平和を持てるのではないだろうか。

『隣人愛』は大小無数のneighborと一体の大きなneighborの集合体である。
小さいneighborと対峙し手で触れることでぬいぐるみとしてだけでなくneighborという存在の温もりや安らぎを感じてもらい、
ぬいぐるみの中に入ることで更に温もりや安心を体感してもらいたい。
快適に過ごすには窮屈な空間ではあるが、その狭さはまるで自分が優しさに包みこまれているような安らぎに繋がると期待する。
大きなぬいぐるみは、neighborという安全基地に自己の精神が守られていることを具象化している。

無数のneighborたちに囲まれた大きなneighborの中に入ることで、彼らの一部と化すことができる。
これは無個性や同化を示唆しているのではない。誰もが心にneighborを持っているとしたら、
隣り合い触れ合うneighborたちの中にも誰かが入っているということなのだ。
本来心の中にあるはずのneighborが触れ合っていることは、心の柔らかい素直な部分が触れ合っているのである。
誰もが優しいということを、感じたいし感じてもらいたい。
