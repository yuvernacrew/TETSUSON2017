出展者の皆さまとスタッフ、OBOGとの交流のためのオープニングパーティーを開催いたします。
通訳スタッフもおりますので、ぜひ日韓交流の機会にもお役立てください！

現在活躍中のデザイナーや建築家、アーティストの方々をお呼びしての作品講評・審査会を行います。

当日正午より、出展者の皆様の作品を、一人一人各審査員様にプレゼンテーションしていただきます。
その際、審査員の方々には簡単な講評も行っていただきます。
こちらの各出展者様の作品の巡回審査を終えたのち、夕方より審査員の方々で公開ディスカッションを行っていただき、
各審査員賞を決定いたします。

TETSUTONの公開審査では、皆様あまり講評していただくことのない、他分野のプロからのご意見をお聞きすることができます。
この機会にぜひご参加くださいませ。なお、全て一般の方々にもご覧いただけますので、みなさまぜひご来場ください！
( 韓国出展者の作品に関しては全て通訳がつきます )

廣村正彰/Masaaki Hiromura

1954年愛知県生まれ。1988年廣村デザイン事務所設立。
グラフィックデザインの他、商業施設や美術館などのサインデザイン、CI、VI計画を多く手がける。
東京工芸大学教授。一般社団法人ジャパンクリエイティブ代表理事。
主な仕事に、日本科学未来館CIサイン計画、横須賀美術館VIサイン計画、9thナインアワーズADサイン計画、
すみだ水族館VIサイン計画、あべのハルカスサイン計画、TOTO MUSEUM VIサイン計画、そごう・西武、
ロフトのアート・ディレクション等。
2015年11月21日より金沢21世紀美術館にて『金沢でJunglin'』展開催
